# GRAPHML - David Agudelo 2026

Colección de Jupyter Notebooks para el estudio y análisis de representaciones gráficas y espaciales aplicadas a la arquitectura y el diseño urbano.

## Contenido

| Notebook | Descripción |
|----------|-------------|
| `S02-01 Primal vs Dual.ipynb` | Comparación entre grafos primales y duales |
| `S02-02 Metric vs Topological.ipynb` | Diferencias entre representaciones métricas y topológicas |
| `S02-03 Adjacency Vs Access.ipynb` | Análisis de adyacencia versus accesibilidad en grafos |
| `S02-04 Geometric Representations.ipynb` | Representaciones geométricas de estructuras espaciales |
| `S02-05 Spatial Representations.ipynb` | Representaciones espaciales y su análisis |
| `S02-06 Importing OBJ files.ipynb` | Importación y visualización de archivos OBJ 3D |

## Archivos de Modelos 3D

El repositorio incluye modelos 3D en formato OBJ/MTL:

- `Test 01.obj / Test 01.mtl`
- `Test 02.obj / Test 02.mtl`
- `Test 03.obj / Test 03.mtl`
- `Test_04.obj / Test_04.mtl`

## Requisitos

- Python 3.x
- Jupyter Notebook / JupyterLab
- Librerías: `numpy`, `matplotlib`, `networkx` (y otras según cada notebook)

## Uso

1. Clona o descarga el repositorio.
2. Abre Jupyter Notebook o JupyterLab.
3. Ejecuta los notebooks en orden (S02-01 → S02-06) para seguir la progresión del material.
